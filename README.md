# Sales Data Analysis Application

Приложение для анализа продаж и статистического исследования данных с GUI и текстовым интерфейсами.

## Описание

Проект фокусируется на Data Science и статистике, без машинного обучения. Приложение позволяет работать с данными, выполнять группировки, рассчитывать базовые статистики и визуализировать данные как текстово, так и графически.

## Основные возможности

### 1. Data Cleaning (Очистка данных)
- Обработка пропусков и некорректных значений
- Конвертация колонок в правильный тип: date → datetime, quantity → int
- Обнаружение и удаление выбросов (IQR метод)

### 2. Descriptive Statistics (Описательная статистика)
- Среднее, медиана, минимум/максимум, стандартное отклонение по `total_amount` и `quantity`
- Возможность расчёта по категориям (`product_category`)

### 3. Aggregation & Grouping (Группировка данных)
- Суммарные продажи по категориям и месяцам
- Топ-N категорий по суммарным продажам

### 4. Text-based Visualization (Текстовая визуализация)
- ASCII-гистограммы для категорий товаров или месяцев
- Таблицы с агрегированными данными

### 5. GUI Visualization (Графический интерфейс)
- Бар-чарты по категориям и месяцам с использованием Matplotlib
- Диаграммы суммарных продаж

## Требования к данным

CSV-файл должен содержать следующие колонки:
- `transaction_id` или `Transaction ID`
- `date` или `Date`
- `customer_id` или `Customer ID`
- `gender` или `Gender`
- `age` или `Age`
- `product_category` или `Product Category`
- `quantity` или `Quantity`
- `price_per_unit` или `Price per Unit`
- `total_amount` или `Total Amount`

Приложение автоматически определяет названия колонок (поддерживает как Title Case, так и snake_case).

## Установка

1. Убедитесь, что у вас установлен Python 3.8+
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

### GUI интерфейс

Запустите графический интерфейс:
```bash
python main.py
```

**Меню:**
- **Preprocessing** - загрузка и очистка данных
- **Descriptive statistics** - описательная статистика
- **Sales by category** - продажи по категориям
- **Sales by month** - продажи по месяцам
- **Show charts** - показать графики
- **ASCII histograms** - ASCII гистограммы
- **Open CLI interface** - открыть текстовый интерфейс

**Параметры:**
- Выбор категории для фильтрации
- Выбор месяца для фильтрации
- Top-N для ограничения количества элементов

### Текстовый интерфейс (CLI)

Запустите текстовый интерфейс:
```bash
python cli_interface.py
```

**Меню:**
1. Load dataset - загрузить CSV файл
2. Show summary statistics - показать описательную статистику
3. Show sales by category - показать продажи по категориям
4. Show sales by month - показать продажи по месяцам
5. Show ASCII histograms - показать ASCII гистограммы
0. Exit - выход

### Data Structures & Algorithms
- Использование словарей и defaultdict для подсчётов и группировок

- Алгоритмическая обработка временных рядов: суммирование по месяцам

- Нормализация названий колонок (поддержка разных форматов)
- Автоматическое определение типов данных
- Обработка пропусков и выбросов
